## Demo Mode

In this hackathon skill, wallet funding is simulated for demonstration purposes. In production, you would:

1. Create a real Base Sepolia wallet
2. Fund with ETH from faucet (for gas)
3. Get testnet USDC from Circle faucet
4. Deploy a credit-tracking contract (optional)
